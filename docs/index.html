Set up GlueOps Deployment

This creates a dedicated GitHub App in your organization that GlueOps CD uses to update image tags in your deployment-configurations repo. You must be an owner of the organization.

Must be globally unique and ≤ 34 characters. If too long, shorten it (e.g. glueops-deploy-<org>).

1. Add the App ID (organization variable)

Variable name: GLUEOPS_DEPLOYMENT_APP_ID

Open “New organization variable”

2. Add the private key (organization secret)

Secret name: GLUEOPS_DEPLOYMENT_APP_PRIVATE_KEY

Shown once. This is a sensitive secret — copy it now and treat it like a password. If you reload or leave this page, it is gone for good.

Open “New organization secret”

3. Install the app

Install it and select your deployment-configurations repository — the permissions do nothing until the app is installed there.
